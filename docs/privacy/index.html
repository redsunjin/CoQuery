CoQuery Training

Privacy Policy

Last updated: July 10, 2026

Overview

CoQuery Training is a SQL practice app for iPhone and iPad. The first TestFlight build is Training Mode only and uses bundled sample data.

Data Collection

The first iOS Training build does not collect personal data from the app. It does not require an account, ask for production database credentials, require an API key, or include advertising or tracking SDKs.

Local Device Data

The app may store practice attempts and UI preferences locally on the device so testers can review their own practice flow. This local data is not sent to CoQuery by the first iOS Training build.

Production Databases

The first iOS Training build is not a production database client. It does not connect to production databases and does not request production database credentials.

AI Providers

The first iOS Training build does not require provider-backed AI feedback. If a future build adds provider-backed features, this privacy policy and App Store privacy responses must be reviewed before submission.

Contact

For support, use the contact email configured in the app's TestFlight or App Store listing.

Changes

If app data practices change, this policy and the App Store Connect privacy responses will be updated before submitting a new build.
